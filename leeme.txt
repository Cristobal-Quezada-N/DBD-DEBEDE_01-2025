1-Instalar PostgreSQL
Si aún no lo has hecho, descarga e instala PostgreSQL desde su página oficial.

2-Iniciar PGAdmin 4
Abre la aplicación PGAdmin 4. Esta es la interfaz gráfica que utilizaremos para interactuar con la base de datos.

OPCIONAL: Si prefieres, también puedes usar sqliteOnline en la versión de PostgreSQL.

3-Acceder a la carpeta "Entrega Final"
Dirígete a la carpeta donde tienes los archivos proporcionados, llamada "Entrega Final".

4-Ejecutar el archivo Tablas.sql
Abre Tablas.sql en PGAdmin y ejecútalo.

5-Ejecutar el archivo Inserts.sql
Después de ejecutar las tablas, abre Inserts.sql y ejecútalo.

6-Ejecutar el archivo Sentencias.sql
Finalmente, abre Sentencias.sql.
Es importante ejecutar cada consulta una por una para asegurar que todo funcione correctamente y que las sentencias se ejecuten sin inconvenientes.

Es IMPORTANTE que sigas estas intrucciones para evitar errores en la ejecucion de la base de datos.
